# Use-case walkthroughs

These walkthroughs are **narrated transcripts**: the tool calls and outputs are realistic, but model responses are illustrative so you can adapt them to your own workloads.

- [Kernel research workflow](./kernel-research.md) — reviews a public-style GPU reduction/matmul kernel using `rlm_init`, grep anchors, `rlm_exec` statistics loops, and recursive hotspot checks.
- [Paper analysis workflow](./paper-analysis.md) — deep-dives a long research paper by section with chunk planning, targeted sub-queries, and evidence-first synthesis.
- [Codebase triage workflow](./codebase-triage.md) — triages a pre-concatenated medium/large code corpus using grep-first discovery and stateful investigation maps.

Use them as templates for host-model prompting and for evaluating future GEPA-optimized tool descriptions.
